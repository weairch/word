# Word War
#### 具備多種模式，可讓多位玩家在多重獨立空間下競爭的英文練習網站
<br>

網址 : [wordwar.online](https://wordwar.online "網址:")

- 用有趣的方式練習英文單字
- 可以在對戰室與其他使用者聊天
- 在遊戲中有即時性的體驗
